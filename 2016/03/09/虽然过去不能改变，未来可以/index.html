虽然过去不能改变，未来可以

发表于 2016-03-09   |   分类于 心得体会   |     |   阅读次数

  转眼间已经工作五年了，大大小小的项目做过不少，也经历过几家公司，刚毕业初入职场的时候对一切新事物充满好奇与激情，毕业一年多的时间里从工作到业余尝试了好几种截然不同的身份，自动化测试、开发、销售、在职硕士，最终决定在最感兴趣的iOS上走下去。现在做iOS开发也有四个年头了，再回过头来看，发现虽然过去不能改变，但未来的确把握在我们自己手里。当然，没有人能预测未来，也许当年跟着复旦的导师一起做量化交易，现在发展更好也说不定（YY五秒钟…），但选择没有对错，就像韩寒《后会无期》里说的，小孩子才看对错，成年人只看利弊，如果能让未来成为自己所期望的未来，那这何尝不是我们最好的选择呢。

过去的我们

  高中的时候大家都过着比较类似的生活，无非就是学霸天天看书励志考一所好的大学，学渣偷偷看小说等着毕业出去“闯荡江湖”，当然，现在几乎全民大学了，至少我当年的高中同学全部上了大学，不过好坏之分。人生的每个阶段都会以截然不同的眼界去看待这个世界，大学之前多数人都是“真正的学生”，不管有没有好好学，但围绕的都是学习的事情。走进大学，会发现昔日的发小，曾经的同窗相互之间差距越来越大，价值观也越来越不同。大学没有了监管，有想法的人会慢慢实现自己一直想做的事，离想成为的人越来越近，当然，堕落的人也在做自己想做的事，女孩逛街，男孩泡吧打游戏，四年下来，有些人成为了创业新星，有些人做着自己想做的事，有些人按照自己的规划走入职场找到一份好工作，还有些人也终究不得不面对现实，在堕落了四年之后匆忙准备面试找一份自己并不想做的工作。

现在的我们

  过去与未来不过是一个相对的概念，就看你想跟什么时候的自己说拜拜，又想塑造一个怎样的自己。把大学时光说成过去的我们不过是对大多数人来说，大学毕业才算真正的独立，无论过去是堕落的还是奋进的，走上社会，我们都要开始独立的生活，要考虑柴米油盐，要考虑成家立业，当然，也要考虑自己的理想，不管是刚毕业还是已经工作了三五年，抑或是业界大牛创业精英，这就是现在的我们。如果你已经成为了你想成为的人，那说明几年甚至十几年前你就与过去的自己划清界限，预订了一个美好的未来。如果你还是现在的你，并且不是很满意，要么你是个安于现状的人，要么就是为想要的未来付出的时间还不够，如果你还是过去的你，那么，是时候醒醒了，别荒废了这一生。

我们的未来

  有些人已经活在“未来”中了，但更多的人还是现在的我们，心中应该有个未来，比如我，我想博览群书，我想头脑发达四肢也发达，我想学会分享，我想做全栈，我想成为摄影大师，我想跟我心爱的妻子一起环游世界……我想要的未来太多太多，也许有人觉得这是天方夜谭，四年前我还在复旦读在职硕士，我想做iOS架构师，我想独立开发自己的应用，我想有一家自己的公司，然后我开始制定计划，看博客看开源代码，码代码看框架，如今，虽然不能自称牛逼的架构师，但至少已经在做着架构的工作，也做过独立开发的工作，也有了自己的公司（虽然没赚到钱，囧…），不是有个10000小时法则么，10000个小时可以让你从一个门外汉成为某个领域的专才，所以要敢想敢做，不可以不过是给自己找的接口。今天算是用这篇文章给自己立一个flag吧，未来已经有了，期待实现的那一天。Fighting!!!
